# NotoriousTrack

Elite productivity and habit tracking SaaS web app.

## Tech Stack
- **Frontend**: Next.js 16 (App Router), TypeScript, Tailwind CSS 4, Framer Motion
- **UI Components**: Radix UI, Lucide Icons, Custom Glassmorphism System
- **Charts**: Recharts
- **Styling**: Premium Dark Mode, Indigo-Blue-Cyan Gradients, Glassmorphism

## Key Features
- **Main Dashboard**: Real-time productivity and focus analytics.
- **Habit Tracker**: 31-day visual grid with streak tracking.
- **Biometric Trackers**: Sleep trends, Mood heatmaps, and Hydration (animated progress).
- **Productivity Tools**: Kanban-style Task management and Markdown Journaling.
- **AI Insights**: Behavior analysis and productivity coaching.
- **Gamification**: XP system and Achievement badges.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design Philosophy
Inspired by Linear, Notion, and Stripe Dashboard. Focuses on "Discipline Today. Freedom Tomorrow." through high-quality UI/UX and data-driven insights.
